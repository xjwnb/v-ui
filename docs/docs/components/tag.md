  ## vp-tag

| 属性名             | 类型    | 属性值                                            | 描述             | 默认值    |
| ------------------ | ------- | ------------------------------------------------- | ---------------- | --------- |
| type               | String  | "default"\|"success"\|"info"\|"warning"\|"danger" | 标签类型         | "default" |
| closable           | Boolean | true \| false                                     | 是否可关闭       | false     |
| disableTransitions | Boolean | true \| false                                     | 是否取消关闭动画 | false     |
| effect             | String  | "light" \| "dark"                                 | 主题             | "light"   |
