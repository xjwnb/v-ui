  ## vp-checkbox-group （要和 vp-checkbox 一起用）

| 属性名  | 类型  | 属性值 | 描述               | 默认值 |
| ------- | ----- | ------ | ------------------ | ------ |
| v-model | Array |        | 多选框内容双向绑定 | []     |

  ## vp-checkbox

| 属性名   | 类型    | 属性值        | 描述               | 默认值 |
| -------- | ------- | ------------- | ------------------ | ------ |
| v-model  | Boolean | true \| false | 多选框内容双向绑定 | false  |
| label    | String  |               | 多选框标签内容     | ""     |
| disabled | Boolean | true \| false | 是否禁用           | false  |
